==============
italian.cities
==============

Dummy Plone Dexterity content type to show a problem with plone.formwidget.autocomplete's AutocompleteFieldWidget.

Installation
------------

1. `git clone git@github.com:davide-targa/italian.cities.git`
2. `cd italian.cities`
3. `./bootstrap.sh`
4. `bin/instance fg`
